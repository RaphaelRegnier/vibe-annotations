# Workflows

Choose the workflow that fits your use case:

- [**Copy & paste**](/docs/workflows/copy-paste) — Quick edits on a single page. No server needed.
- [**Multi-page (MCP)**](/docs/workflows/mcp) — Annotate across routes, let your agent read them all via MCP.
- [**Watch mode**](/docs/workflows/watch-mode) — Hands-free loop: drop annotations, agent implements them live.
- [**Collaboration**](/docs/workflows/collaboration) — Export annotations and share with teammates via JSON files.
- [**Agent annotation**](/docs/workflows/agents) — Let browser-capable AI agents annotate for you.
