# Workflows

Choose the workflow that fits your use case:

- [**Copy & paste**](/docs/workflows/copy-paste) — Quick edits on a single page. No server needed.
- [**Multi-page (MCP)**](/docs/workflows/mcp) — Annotate across routes, let your agent read them all via MCP.
- [**Watch mode**](/docs/workflows/watch-mode) — Hands-free loop: drop annotations, agent implements them live.
- [**Design edits**](/docs/workflows/design-edits) — Tweak size, spacing, color and text with live preview.
- [**Variant generation**](/docs/workflows/variants) — Ask your agent for component variants, preview and pick the winner.
- [**Screenshots & images**](/docs/workflows/screenshots) — Auto-capture the element and attach your own reference images.
- [**Sharing exports**](/docs/workflows/sharing) — Package a batch as one `.md` or self-contained `.html` file.
- [**Collaboration**](/docs/workflows/collaboration) — Round-trip annotations to a teammate's localhost.
- [**Agent annotation**](/docs/workflows/agents) — Let browser-capable AI agents annotate for you.
